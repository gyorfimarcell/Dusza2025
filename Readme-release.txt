A program a Cluster.exe fájl futtatásával indítható.

A tesztelés megkönnyítésének érdekében a "Logs" mappában található néhány napló, amit a program automatikusan betölt.
A "test_clusters" mappában található néhány minta klaszter is.

A dokumentáció elérhető a https://gyorfimarcell.github.io/Dusza2025/ linken.
Ha az oldal valami miatt elérhetetlenné válna, a dokumentáció offline példánya elérhető a "dokumentacio-offline.zip"
kicsomagolásával és az "index.html" fájl megnyitásával.
